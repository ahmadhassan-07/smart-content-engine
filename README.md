# 🤖 AI Smart Content Moderator

A Multi-Modal AI web app for automated content moderation and analytics.

## Features
- 📸 **Computer Vision** — YOLOv8 object detection & safety check
- 🧠 **Sentiment Analysis** — DistilBERT transformer
- 🕵️ **Fake Review Detector** — URL scraping + pattern analysis
- 🔗 **Link Analyzer** — Amazon, Daraz, eBay, Facebook, Instagram & more
- 📊 **Engagement Predictor** — ML-based scoring
- ✍️ **Caption Generator** — Auto hashtags & captions

## Tech Stack
Python · Streamlit · YOLOv8 · HuggingFace Transformers · BeautifulSoup · PyTorch · OpenCV

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Supported Platforms
Amazon · Daraz · Shopify · eBay · Facebook · Instagram · TikTok · Snapchat
